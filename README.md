# Easy Chinese Recipes

A clean, static, SEO-friendly Chinese recipe website for beginners and overseas home cooks.

## Features
- Home page with popular and latest recipes
- Category filtering page
- Single recipe detail page
- Random recipe picker
- SEO meta tags
- Recipe JSON-LD schema
- Responsive design
- Pure HTML + CSS + JavaScript
- Ready for GitHub Pages
